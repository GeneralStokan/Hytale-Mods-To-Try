## Contributing

Thanks for helping grow the Hytale mods list. Pull requests are the fastest way to add a mod. But you can also start a discussion or join the created one to request a mod

## How To Submit A Pull Request
1. Fork this repository.
2. Create a new branch for your change.
3. Edit the README and add your mod to the best matching section.
4. Add a short, clear description and a direct link to the mod page.
5. Check that the table formatting stays aligned and the link works.
6. Open a pull request and include a brief summary of what you added.

## Mod Entry Guidelines
- Use the format: Mod name (Author) in the Mod column.
- Keep descriptions short, factual, and free of spoilers.
- Prefer official download or project pages.
- Avoid duplicate entries and check for existing alternatives.
